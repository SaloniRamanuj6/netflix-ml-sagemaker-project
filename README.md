# Netflix User Rating Prediction & Recommendation System (AWS SageMaker)

## Overview
This project builds an end-to-end machine learning pipeline to analyze Netflix content and predict user ratings and engagement behavior using AWS SageMaker.

## Features
- Data preprocessing and cleaning
- Feature engineering (watch_time, watch_ratio)
- Encoding of categorical features
- XGBoost regression and classification models
- Deployment using SageMaker endpoint
- Monitoring with AWS CloudWatch

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, XGBoost, AWS SageMaker

## Results
- Classification Accuracy: ~56%
- Successfully deployed real-time endpoint
